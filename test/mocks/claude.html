Building a Chrome extension

How do I create a Chrome extension that syncs data across tabs?
Claude

I'll help you create a Chrome extension for syncing data across tabs. Here's the approach:

  1. Use chrome.storage.local for persistent storage
  2. Set up message passing between content scripts and background
  3. Use chrome.tabs API to communicate across tabs
Human
Can you show me a code example?
Claude

Here's a basic example:


// background.js
chrome.runtime.onMessage.addListener((request, sender, sendResponse) => {
  if (request.type === 'SYNC_DATA') {
    chrome.storage.local.set({ data: request.data });
    sendResponse({ success: true });
  }
  return true;
});
