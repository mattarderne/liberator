Machine learning basics

Explain supervised learning in simple terms

Supervised learning is like learning with a teacher! Here's how it works:

  • Training data: You give the model examples with correct answers
  • Learning: The model finds patterns in the data
  • Prediction: It uses those patterns to answer new questions

Think of it like studying flashcards - you see the question and answer together, then later you can answer similar questions on your own.

Can you show me a simple Python example?

Here's a basic supervised learning example using scikit-learn:


from sklearn.linear_model import LinearRegression
import numpy as np

# Training data (X = features, y = labels)
X = np.array([[1], [2], [3], [4], [5]])
y = np.array([2, 4, 6, 8, 10])

# Create and train the model
model = LinearRegression()
model.fit(X, y)

# Make a prediction
prediction = model.predict([[6]])
print(f"Prediction for 6: {prediction[0]}")  # Output: 12
